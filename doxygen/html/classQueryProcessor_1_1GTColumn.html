Detailed Description

This class stores and processes information about a single column of the gVCF array. A column corresponds to a unique position in the genome. It holds the REF, ALT and PL values of every row (i.e., individual) in the array. If a REF value is equal to "$" for a row, then it means that this row is NULL (i.e., it contains no useful info for joint genotyping).
