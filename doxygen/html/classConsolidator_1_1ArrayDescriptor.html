Detailed Description

This class is a wrapper for a ArrayInfo object. It is returned by Consolidator::open_array, and used for the various functions of the consolidator. It contains a pointer to an ArrayInfo object in Consolidator::open_arrays_, along with ArrayDescriptor::array_info_id_ that is used for debugging purposes to check if the stored ArrayInfo object is obsolete (i.e., if it has been deleted by the consolidator when closing the array).
