13 // Functions: LinetoAxes
14 // AxestoLine
15 //
16 // Purpose: Serial Hilbert length <----> multidimensional Axes position.
17 //
18 // Space = n-dimensional hypercube of side R = 2^b
19 // Number of cells = N = R^n = 2^(n*b)
20 //
21 // Line = serial number of cell along Hilbert curve through hypercube
22 // = extended integer of n*b bits ranging from 0 to N-1,
23 // stored as vector of n unsigned b-bit integers with [0] high.
24 //
25 //
26 // A composite-integer is a multi-word unsigned integer "Label" stored
27 // "big endian" in N conventional unsigned integers with [0] high.
28 // ___________________________________________________
29 // | | | | |
30 // | Label[0] | Label[1] | .... | Label[N-1] |
31 // |____________|____________|____________|____________|
32 // high low
33 //
34 // Axes = Geometrical position of cell
35 // = n b-bit integers representing coordinates.
36 //
37 // Example: side R = 16, dimension n = 2, number of cells = N = 256.
38 // Line = 9, stored in base-16 words as
39 // Line[0] = 0 (high), Line[1] = 9 (low),
40 // corresponds to position (2,3) as in diagram, stored as
