Detailed Description

This struct groups info about an array fragment (e.g., schema, book-keeping structures, etc.).

Member Data Documentation

const ArraySchema* StorageManager::FragmentInfo::array_schema_

The array schema (see class ArraySchema).

BoundingCoordinates StorageManager::FragmentInfo::bounding_coordinates_

Stores the bounding coordinates of every (coordinate) tile, i.e., the first and last cell of the tile (see Tile::bounding_coordinates).

Mode StorageManager::FragmentInfo::fragment_mode_

The fragment mode (see StorageManager::Mode).

std::string StorageManager::FragmentInfo::fragment_name_

The fragment name.

int64_t StorageManager::FragmentInfo::id_

Unique FragmentInfo object id, for debugging purposes when using StorageManager::FragmentDescriptor objects (see StorageManager::FragmentDescriptor::fragment_info_id_).

std::vector<int64_t> StorageManager::FragmentInfo::lastly_appended_tile_ids_

It keeps the id of the lastly appended tile for each attribute. It is used for debugging purposes to ensure the fragment "correctness" in StorageManager::check_on_append_tile.

MBRs StorageManager::FragmentInfo::mbrs_

Stores the MBR of every (coordinate) tile.

Offsets StorageManager::FragmentInfo::offsets_

Stores the offset (i.e., starting position) of every tile of every attribute in the respective data file.

PayloadSizes StorageManager::FragmentInfo::payload_sizes_

Stores the aggregate payload size of the tiles currently stored in main memory for each attribute.

PosRanges StorageManager::FragmentInfo::pos_ranges_

Stores the range of the position of the tiles currently in main memory, for each attribute. The position of a tile is a sequence number indicating the order in which it was appended to the fragment with respect to the the other tiles appended to the fragment for the same attribute (e.g., 0 means that it was appended first, 1 second, etc.).

Segments StorageManager::FragmentInfo::segments_

Stores one segment per attribute.

TileIds StorageManager::FragmentInfo::tile_ids_

Stores all the tile ids of the fragment.

Tiles StorageManager::FragmentInfo::tiles_

Stores the tiles of every attribute currently in main memory.
