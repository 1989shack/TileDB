DESCRIPTION

This file defines struct TileDB_Context, which constitutes the TileDB state that wraps the various TileDB modules. It also declares the context initialization/finalization functions of TileDB.

TILEDB_EXPORT int tiledb_init ( const char *  workspace,
TileDB_Context *&  tiledb_context 
)

Initializes the TileDB context. On error, it prints a message on stderr and returns an error code (shown below).

Parameters
workspaceThe path to the workspace folder, i.e., the directory where TileDB will store the array data. The workspace must exist, and the caller must have read and write permissions to it.
tiledb_contextThe TileDB state.
Returns
An error code, which can be one of the following:
  • 0
    Success
  • TILEDB_ENSMCREAT
    Failed to create storage manager
  • TILEDB_ENLDCREAT
    Failed to create loader
  • TILEDB_ENQPCREAT
    Failed to create query processor
