Detailed Description

This struct groups info about an array (e.g., schema, book-keeping structures, etc.).

Member Data Documentation

The array mode (see StorageManager::ArrayMode).

ArraySchema StorageManager::ArrayInfo::array_schema_

The array schema (see class ArraySchema).

BoundingCoordinates StorageManager::ArrayInfo::bounding_coordinates_

Stores the bounding coordinates of every (coordinate) tile, i.e., the first and last cell of the tile (see Tile::bounding_coordinates).

std::string StorageManager::ArrayInfo::fragment_name_

The fragment name.

uint64_t StorageManager::ArrayInfo::id_

Unique ArrayInfo object id, for debugging purposes when using StorageManager::ArrayDescriptor objects (see StorageManager::ArrayDescriptor::array_info_id_).

std::vector<uint64_t> StorageManager::ArrayInfo::lastly_appended_tile_ids_

It keeps the id of the lastly appended tile for each attribute. It is used for debugging purposes to ensure the array "correctness" in StorageManager::check_on_append_tile.

MBRs StorageManager::ArrayInfo::mbrs_

Stores the MBR of every (coordinate) tile.

Offsets StorageManager::ArrayInfo::offsets_

Stores the offset (i.e., starting position) of every tile of every attribute in the respective data file.

PayloadSizes StorageManager::ArrayInfo::payload_sizes_

Stores the aggregate payload size of the tiles currently stored in main memory for each attribute.

RankRanges StorageManager::ArrayInfo::rank_ranges_

Stores the range of the ranks of the tiles currently in main memory, for each attribute. The rank of a tile is a sequence number indicating the order in which it was appended to the array with respect to the the other tiles appended to the array for the same attribute (e.g., 0 means that it was appended first, 1 second, etc.).

TileIds StorageManager::ArrayInfo::tile_ids_

Stores all the tile ids of the array.

Tiles StorageManager::ArrayInfo::tiles_

Stores the tiles of every attribute currently in main memory.
