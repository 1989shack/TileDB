\subsection{Detailed Description}
In our external sorting algorithm, a run is a sorted sequence of cells, created during the 'sort' phase. Subsequently, multiple runs are merged into a single one during the 'merge' phase, potentially recursively. A \hyperlink{classStorageManager_1_1Run}{Run} object stores information about a sorted run.
