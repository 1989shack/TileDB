\subsection{Detailed Description}
This class is a wrapper for a \hyperlink{structConsolidator_1_1ArrayInfo}{Array\+Info} object. It is returned by \hyperlink{classConsolidator_afce022da3083ef724772deda811d8924}{Consolidator\+::open\+\_\+array}, and used for the various functions of the consolidator. It contains a pointer to an \hyperlink{structConsolidator_1_1ArrayInfo}{Array\+Info} object in \hyperlink{classConsolidator_a3a86f55ce360cefaa78c89fead0e96d8}{Consolidator\+::open\+\_\+arrays\+\_\+}, along with \hyperlink{classConsolidator_1_1ArrayDescriptor_a6c4666c014b3f10145e678e1a3fb2de0}{Array\+Descriptor\+::array\+\_\+info\+\_\+id\+\_\+} that is used for debugging purposes to check if the stored \hyperlink{structConsolidator_1_1ArrayInfo}{Array\+Info} object is obsolete (i.\+e., if it has been deleted by the consolidator when closing the array).
