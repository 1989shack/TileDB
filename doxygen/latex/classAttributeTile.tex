\subsection{Detailed Description}
\subsubsection*{template$<$class T$>$class Attribute\+Tile$<$ T $>$}

This class defines an attribute tile. Attribute tiles consist of a collection of cells of template type T, which are stored in vector \hyperlink{classAttributeTile_a1fc1096adebca025565e774a89937736}{Attribute\+Tile\+::payload\+\_\+}.
